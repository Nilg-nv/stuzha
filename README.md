# stuzha
Игра посвещена выживанию в тяжелый условиях дальнего севера

## Общее описание
- жанр - песочница, выживание
- от 1-го лица
- 2D графика
- камера сверху

## План
- разработка простого квадрата на экране
- собрать под wasm
- разработка двигающегося квадрата
- движение в зависимости от нажатий
- движение пиксельной картинки
- добавление звука при нажатии на клавишу L
- сделать простую ходилку со сбором предметов
- добавить случайный разброс предметов
- добавить плитки для карты
- добавить влияние плиток на скорость

## Движок 
На основе проекта будет формироваться движок [nura](https://github.com/Nilg-nv/nura) 

## Библиотеки 
- для графики [gogpu/gg](https://github.com/gogpu/gg)
- для аудио
    - временно [ebitengine/oto](https://github.com/ebitengine/oto)
    - как будет работать кросплатформа [gogpu/audio](https://github.com/gogpu/audio)

## Для максимальной эффективности в хранении звука стоит использовать формат Opus
если достаточно моно, можно сэкономить ещё места
- 48 кбит/с - фоновые звуки (шаги, шелест, ветер, дождь)
- 64 кбит/с - отдельные звуки (удары, выстрелы, пикапы, уведомления)
- 80 кбит/с - голос (диалоги)
- 128 кбит/с - музыка 
- 160 кбит/с - практически соотствует несжатому
